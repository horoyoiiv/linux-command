

# who  

* 현재 로그인된 유저 정보 확인  

### 경로  

* /usr/bin/who : 실행 파일 경로  
* /var/run/utmp : 해당 파일에서 정보를 읽어온다.  

### 옵션  

-H : 헤더 정보 함께 출력  
-a : 모든 정보 출력  
-r : run level을 출력  

### 출력  
* 유저 이름  
* pts?  
* 로그인한 시각
